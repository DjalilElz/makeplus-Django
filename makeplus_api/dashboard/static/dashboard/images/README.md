# Logo Placement

## Place your MakePlus logo here

**Required file:** `makeplus-logo.png`

**Location:** `dashboard/static/dashboard/images/makeplus-logo.png`

**Recommended specifications:**
- Format: PNG (with transparent background recommended)
- Dimensions: Height should be around 100-120px (width will scale proportionally)
- File name: `makeplus-logo.png` (exact name required)

The logo will be displayed at the top of both:
- Admin/Staff sidebar
- Committee Member sidebar

The logo height is set to 40px in the template and will scale proportionally to maintain aspect ratio.

## Alternative formats supported:
- `makeplus-logo.png` (recommended)
- `makeplus-logo.jpg`
- `makeplus-logo.svg`

If using a different format, update the file extension in `base.html` template.
